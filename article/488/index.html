Langchain の SQL Database Toolkits の AgentExecutor で自然言語からSQLを生成しデータベースにアクセスする

2024-05-26 llm

Langchain の SQL Database Toolkits は SQL を生成しデータベースにアクセスする機能を提供する。

create_sql_agent() は SQLDatabaseToolkit.get_tools() の

  • query_sql_database_tool: SQLを実行する
  • info_sql_database_tool: テーブルのスキーマを取得する
  • list_sql_database_tool: テーブル名の一覧を取得する
  • query_sql_checker_tool: SQLが誤っている場合は修正する

を tools として追加しプロンプトに埋めるなどした LLM の AgentExecutor を返す。

LangChain のエコシステム LangGraph で cycle を含むグラフから LLM のエージェントを作る - sambaiz-net

ドキュメントと同じく Chinook database で実行してみる。

from langchain_community.agent_toolkits import create_sql_agent
from langchain_openai import ChatOpenAI
from langchain_community.utilities.sql_database import SQLDatabase

db = SQLDatabase.from_uri("sqlite:///Chinook_Sqlite.sqlite")
llm = ChatOpenAI(model="gpt-4-turbo", temperature=0)
agent_executor = create_sql_agent(llm, db=db, agent_type="openai-tools", verbose=True)
result = agent_executor.invoke(
    "List the total sales per country. Which country's customers spent the most?"
)
print(result)

verbose=True で実行しているので過程も出力される。テーブル名の一覧から Customer と Invoice テーブルのスキーマを取得し JOIN するクエリを生成して データを取得していることが確認できる。メタデータもなしに必要なテーブルだけがピンポイントで取得できているが、これはある程度一般的な命名とテーブル構造に基づいているものだと考えている。

# Who is the most valuable customer in this company? Why? What did they buy the most?
Helena Holý's most frequently purchased item is the track "New Year's Day," which she bought once.

In summary:
- The most valuable customer is Helena Holý, who has spent a total of $49.62.
- The product she bought the most is the track titled "New Year's Day."

SQLDatabase は内部で SQLAlchemy を使っているので sqlalchemy.dialects を提供する PyAthena のようなライブラリをインストールすることで様々なデータソースに接続できる。

s3_staging_dir = "s3://<bucket>"
database = "<database>"
conn_str = f"awsathena+rest://@athena.ap-northeast-1.amazonaws.com:443/{database}?s3_staging_dir={s3_staging_dir}"
db = SQLDatabase.from_uri(conn_str)
print('execute', db.run("SELECT * FROM <database>.<table> LIMIT 10"))
print('get_usable_table_names', db.get_usable_table_names())
print('get_table_info', db.get_table_info(table_names=['test']))
