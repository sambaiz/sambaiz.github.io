CDKがECRへのpushまでやってくれるのでcdk deployすれば動き始め、削除するときもStackを消せばよい。

$ cat Makefile
.PHONY: deploy destroy

deploy:
	cd cdk && npm install && npm run build && npm run cdk -- deploy

destroy:
	cd cdk && npm install && npm run cdk -- destroy
