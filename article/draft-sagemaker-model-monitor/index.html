SageMaker Model Monitorでデプロイしたモデルの品質を監視する

2020-07-26 pythonpytorchmachinelearningaws

SageMaker Model Monitorは デプロイしたモデルの入出力データをキャプチャして品質を監視する機能。

SageMakerで学習したPyTorchのモデルをElastic Inferenceを有効にしてデプロイする - sambaiz-net

入出力データのキャプチャ

deploy()のdata_capture_configで割合と保存先を設定する。

from sagemaker.model_monitor import DataCaptureConfig

data_capture_config=DataCaptureConfig(
                        enable_capture = True,
                        sampling_percentage=100,
                        destination_s3_uri=s3_capture_upload_path)

predictor = model.deploy(..., data_capture_config=data_capture_config)

ベースラインの作成

監視する際の基準はどうするかというと、suggest_baseline()を呼ぶと Spark上で動くDeepuというOSSで学習データからベースラインを決定してくれる。

入力となるCSVを作る。出力が先頭か末尾のカラムに来るようにする。

import csv
dataset_train = datasets.MNIST('mnist', train=True, transform=transforms.ToTensor(), download=False)
os.makedirs('monitor-dataset', exist_ok=True)
with open('monitor-dataset/training-dataset-with-header.csv', 'w') as f:
    writer = csv.writer(f)
    writer.writerow(['label', 'data'])
    for data, target in dataset_train:
        writer.writerow([[target], data.numpy().tolist()])
        
sagemaker.Session().upload_data(path='monitor-dataset', bucket=os.environ["DATASETS_BUCKET"], key_prefix='monitor-dataset')

$ cat model-monitor-preprocess.py
import torch
import json

def preprocess_handler(inference_record):
    input_data = inference_record.endpoint_input.data
    output_data = inference_record.endpoint_output.data

    input_data['data'] = torch.FloatTensor(json.load(input_data['data']))
    output_data['label'] = json.load(output_data['label'])
    
    return {**input_data, **output_data}

監視を始める

from sagemaker.model_monitor import CronExpressionGenerator
from time import gmtime, strftime

mon_schedule_name = 'DEMO-xgb-churn-pred-model-monitor-schedule-' + strftime("%Y-%m-%d-%H-%M-%S", gmtime())
my_default_monitor.create_monitoring_schedule(
    schedule_name=mon_schedule_name,
    endpoint_input=predictor.endpoint,
    post_analytics_processor_script=s3_code_postprocessor_uri,
    output_s3_uri=s3_report_path,
    statistics=my_default_monitor.baseline_statistics(),
    constraints=my_default_monitor.suggested_constraints(),
    schedule_cron_expression=CronExpressionGenerator.hourly(),
    enable_cloudwatch_metrics=True,
)
