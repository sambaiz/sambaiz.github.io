OpenAIのGPTを国会会議録の総理大臣の発言でファインチューニングする

2023-09-11 pythonmachinelearningllm

OpenAI は GPT による会話文の生成や 文章をベクトルに変換するほかに 独自のデータセットによってモデルをファインチューニングする API を提供しており、 これを用いることで出力の質を上げたり few-shot learning の例示にかかるコストを節約したりすることができる。

# !pip install openai
import openai
import numpy as np

response = openai.ChatCompletion.create(
  model="gpt-3.5-turbo",
  messages=[{
    "role": "user", # "system", "user", or "assistant"
    "content": "Is this a pen?"
  }],
  temperature=0.5,
)

print(response.choices[0].message)
'''
{
  "role": "assistant",
  "content": "As an AI, I don't have the ability to see or perceive objects. Therefore, I cannot determine if something is a pen or not based on visual cues."
}
'''

embedding = openai.Embedding.create(
  model="text-embedding-ada-002", # https://platform.openai.com/docs/guides/embeddings/embedding-models
  input=response.choices[0].message.content
)

print(embedding.data[0].embedding[:5]) # [-0.02444140985608101, -0.015481687150895596, 0.009029921144247055, 0.0035672972444444895, -0.006072063464671373]
print(len(embedding.data[0].embedding)) # 1536

# https://platform.openai.com/docs/guides/embeddings/limitations-risks
# https://github.com/openai/openai-python/blob/5d50e9e3b39540af782ca24e65c290343d86e1a9/openai/embeddings_utils.py#L65C1-L65C1
def cosine_similarity(a, b):
  return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

cosine_similarity(embedding.data[0].embedding, embedding.data[0].embedding) # 1.0

データセットの準備

国会会議録検索システムのAPIで総理大臣または代理への質問およびその回答を集めて、 ファインチューニング用のフォーマットに加工して出力する。 発言の中には非常に長い質問や回答もあるが、主に学習コストのために除外している。 後で気づいたがJSONで取得することもできる。

# !pip install beautifulsoup4
import requests
from bs4 import BeautifulSoup
import re
import json
from datetime import date, timedelta

p = re.compile('〔[^〔〕]*〕')
with open("fine_tune.jsonl", "w") as file:
  day_until = date(2021, 12, 31)
  for i in range(6):
    day_from = day_until + timedelta(days=1)
    day_until = day_from + timedelta(days=30)
    print(f"{day_from.strftime('%Y-%m-%d')} - {day_until.strftime('%Y-%m-%d')}")
    params = {
      'any': 1,
      'maximumRecords': 10,
      'from': day_from.strftime('%Y-%m-%d'),
      'until': day_until.strftime('%Y-%m-%d')
    }
    response = requests.get('https://kokkai.ndl.go.jp/api/meeting', params=params)
    data = response.text

    soup = BeautifulSoup(data, 'html.parser')
    speechrecords = soup.find_all('speechrecord')
    for i, record in enumerate(speechrecords):
      if record.speaker.text == '会議録情報':
        continue
      # 内閣総理大臣臨時代理・厚生大臣, 副議長 etc.
      if record.speakerposition.text.startswith('内閣総理大臣')
              and i != 0 and not speechrecords[i-1].speakerposition.text.endswith('議長'):
        messages = [{'role': 'system', 'content': 'あなたは総理大臣です。'}]
        for j in reversed(range(2)):
          if len(speechrecords[i-j].speech.text) > 200:
            break
          role = ['assistant', 'user'][j%2]
          text = '　'.join(speechrecords[i-j].speech.text.split('　')[1:]) # remove speaker name
          text = p.sub('', text) # remove actions such as〔賛成者起立〕
          text = ''.join(text.split()).replace('――――◇―――――', '') # remove line breaks and etc.
          messages.append({"role": role, "content": text})
        if len(messages) == 3:
          file.write(json.dumps({"messages": messages}, ensure_ascii=False) + '\n')

ファインチューニング

OpenAI Python Library で ファイルをアップロードし ファインチューニングの API を呼び出す。

import openai
import time

with open("fine_tune.jsonl", "r") as file:
  upload = openai.File.create(
    purpose="fine-tune",
    file=file,
  )

while upload.status != "processed": # uploaded, processed, pending, error, deleting or deleted
  time.sleep(5)
  upload = openai.File.retrieve(id=upload.id)
  print(f"file status: {upload.status}")

job = openai.FineTuningJob.create(
  model="gpt-3.5-turbo",
  suffix='souri',
  training_file=upload.id
)

while job.status != "succeeded": # created, running, succeeded, failed, or cancelled
  time.sleep(10)
  job = openai.FineTuningJob.retrieve(id=job.id)
  print(f"fine-tunes job status: {job.status}")

ジョブが終了するとメールが届くが、CLIでステータスなどを確認することもできる。22分ほどかかった。

会話文の生成

API を呼ぶこともできるが、色々試すには Playground が便利。

まずは元モデルで"あなたは総理大臣です"という指示をして会話したところ一般的な質問に対しては通常のように回答してしまった。

ファインチューニング後のモデルでも試したところ最後まで国会での答弁のような回答をすることができた。 データセットの文章の長さを制限したためか簡潔な文章が返ってくる。

モデルの評価には OpenAI Evals を用いることもできる。
